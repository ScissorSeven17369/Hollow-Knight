HOW TO IMPORT HOLLOW KNIGHT SPRITES

1. Using Spritesheets

Create a sprites/ folder. Place spritesheet PNGs with evenly spaced frames.

// In script.js, update SPRITE_CONFIG:
SPRITE_CONFIG.player.idle = {
    src: 'sprites/knight-idle.png',
    frames: 11,
    frameWidth: 120,
    frameHeight: 120,
    frameRate: 8
};

2. Using Individual PNGs

Place individual frames in folders:

sprites/idle/frame0.png
sprites/idle/frame1.png
sprites/run/frame0.png
sprites/run/frame1.png
// Use 'folder' mode:
SPRITE_CONFIG.player.idle = {
    mode: 'folder',
    src: 'sprites/idle/',
    files: ['frame0.png','frame1.png','frame2.png'],
    frameWidth: 120,
    frameHeight: 120,
    frameRate: 8
};

3. Loading Animations

Animations are loaded at game start. Add new animation states:

// Add to animationDefinitions:
'wall_slide': { spriteKey: 'wall_slide', frames: 6, frameRate: 10 }
'wall_jump': { spriteKey: 'wall_jump', frames: 4, frameRate: 12 }

4. Changing Skins

// Replace the entire spritesheet source:
SPRITE_CONFIG.player.idle.src = 'sprites/great-nail-idle.png';
// Then reload the game.

5. Sprite Sources

  • Hollow Knight sprite rips from The Spriters Resource
  • Custom art at 120x120px per frame
  • Transparent PNG backgrounds
  • Recommended: 24fps sprite sheets
